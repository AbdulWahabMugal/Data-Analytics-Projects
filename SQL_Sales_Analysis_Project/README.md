Project: SQL Sales & Business Performance Analysis

Objective:
Analyze sales and profit data using SQL to generate business performance insights and KPI reports.

Dataset:
Orders.csv, Details.csv (Transactional Sales Data)

Tools Used:
SQL (SQLite)

Work Performed:

Calculated total sales KPI using aggregation functions

Analyzed category-wise sales and profit performance

Performed state-wise sales analysis using JOIN between Orders and Details tables

Used GROUP BY and ORDER BY for performance reporting

Business Outcome:
Enabled structured sales reporting, category performance analysis, and region-wise revenue insights to support data-driven decision-making.
